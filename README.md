# Health Insurance Claims Analytics

## Project Overview
This project analyzes health insurance claims to identify factors driving high medical costs.

## Business Problem
Insurance companies need to understand what drives high claims in order to manage risk.

## Dataset
Insurance claims dataset with variables including:
- Age
- BMI
- Smoking status
- Region
- Charges

## Tools Used
- Power BI
- DAX
- Excel

## Analysis Steps
1. Data Cleaning
2. Exploratory Data Analysis
3. Dashboard Development
4. Risk Segmentation

## Dashboard
![Dashboard](images/dashboard_preview.png)

## Key Insights
- Smokers have significantly higher medical expenses.
- BMI correlates strongly with insurance charges.
- Adults aged 45+ generate the highest claims.

## Recommendations
- Adjust premium pricing for high-risk groups.
- Promote wellness programs to reduce claims.
